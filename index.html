Computation-Geometry

An C++ based computation geometry library, used to find the relationship between geometry shapes. For details of implemented algorithms, check the github pages generated by doxygen:

https://maosong2022.github.io/Computation-Geometry/index.html

Usage

To use this repository, first clone the repository:

git clone https://github.com/MaoSong2022/Computation-Geometry.git
cd Computation-Geometry/

then, navigate the src/main.cpp to view how to use algorithms. You may modify the main.cpp to do something.

After modification, use CMake to build the project:

# in root directory
mkdir build && cd build
cmake ..
cmake --build .

after building the project, you can run the program:

# in build directory
./src/Computation_Geometry_run

Test

We use googletest as our test framework. To check the test results, after building the project, run

# in build directory
ctest

to see the overall results.

For details of the project, run

# in build directory
./test/Computation_Geometry_test

to check the details of each test.
